P & NP

  • P: class of problems solvable in polynomial time. O(nkn^kn​k​​) for some constant k.

    Shortest paths in a graph.

  • NP: class of problems verifiable in polynomial time.

​ Verifying a cycle is hamiltonian.

  • NP-complete: problem is in NP and is as hard as any problem in NP.

    如果NPC问题可在多项式时间内被解决，那么所有的NP问题都可以在多项式时间内被解决

Interval Scheduling / 区间调度问题

Requests 1,2,...,n1,2, . . . , n1,2,...,n, single resource

s(i)s(i)s(i) start time, f(i)f(i)f(i) finish time, s(i)<f(i)s(i) < f(i)s(i)<f(i) (start time must be less than finish time for a request)

Two requests iii and jjj are compatible if they don’t overlap, i.e., f(i) ≤ s(j) or f(j) ≤ s(i).

Goal: Select a compatible subset of requests of maximum size.
